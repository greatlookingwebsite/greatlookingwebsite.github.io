Title of page

Smaller line

  • html - defines the root of the html document
  • head - contains html document header information
  • meta name, content, + - information about the html document - read by search engines
  • link - links to document used by calling html document - css files uses this a lot
  • div - defines a section within html document
  • p - open/close of a paragraph
  • b - bold text
  • i - italic text
  • nav -- navigation links
  • a - href, target,
  • h1, h2, h3, etc - defines headings of varying sizes
  • img - img/contains an image file
  • ol, li - list items
  • footer - define the footer, bottom section of html document
  • script - client-side script usually to handle dynamic UI elements on page
